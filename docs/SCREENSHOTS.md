# 功能截图 | Screenshots

云想衣裳各功能模块的界面展示。截图来自实际运行环境，反映真实的使用体验。

---

## 智能衣柜

管理日常穿着单品，支持图片上传和自动标注。

- 衣柜总览页面，以卡片形式展示所有单品
- 添加衣物时自动提取颜色、风格和类型标签
- 支持批量导入和智能去重
- 衣柜诊断功能，分析衣物构成并给出补充建议

## AI 穿搭推荐

上传一张衣物图片，获得个性化的搭配推荐。

- 上传图片后自动识别衣物属性
- 结合天气、用户画像和衣柜单品生成推荐
- 每条推荐附带相似度评分和推荐理由
- 支持将推荐单品加入购物车

## 图像搜索

以图搜图，快速找到相似商品。

- 上传一张参考图片
- 基于 MobileNetV2 特征向量计算相似度
- 按相似度排序展示搜索结果

## 风格画像

分析衣柜单品，生成个人风格分布报告。

- 基于风格分类模型分析每件单品
- 汇总生成六维风格分布图
- 自动保存画像用于后续推荐优化
- 支持的风格：法式风、复古风、极简风、街头风、甜美风、通勤风

## 时尚顾问

基于本地 LLM 的穿搭建议对话。

- 支持自然语言提问
- 对话上下文连续追踪
- 可随时重置对话重新开始
- 依赖 Ollama 本地运行，数据不离开设备

## 用户认证

完整的用户注册和登录流程。

- 支持邮箱注册和登录
- 密码重置功能（通过邮件链接）
- 个人资料管理（身高、体重、体型、肤色等）
- Session 保护机制
